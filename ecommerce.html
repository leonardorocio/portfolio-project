Projeto Ecommerce

Apresentação:

Neste projeto, está sendo desenvolvida uma API de e-commerce utilizando a linguagem Java com o framework Spring Boot. Além disso, a API tem comunicação com banco de dados MySQL gerenciado pelo ORM Hibernate e autenticação utilizando tokens JWT por meio do Spring Security. O objetivo desse projeto é praticar a construção de APIs com o framework Spring Boot e utilizar Spring Data JPA para praticar o uso de banco de dados com uso de repositórios (DAO), utilizar também o Spring Security para praticar como funciona a segurança e diferentes métodos de autenticação.

Para o consumo dos dados dessa API, também foi desenvolvido um frontend utilizando HTML, CSS e Typescript com o framework Angular, com bibliotecas adicionais como o Toastr e o SweetAlert2. O objetivo da construção do frontend além de consumir os dados produzidos pela API do backend foi de aprender a utilizar um framework frontend moderno, trabalhando com componentes, routers e outros, bem como aprimorar os conhecimentos em HTML, CSS e TypeScript.

Neste projeto também foi utilizado o Docker, onde foi criado um container com suas respectivas dependências para cada parte da aplicação: um para banco de dados, outro para o backend em Spring Boot e mais um para o frontend em Angular. Todas as imagens estão disponíveis no Docker Hub no perfil: https://hub.docker.com/u/leonardorocio.

Algumas funcionalidades do sistema:

  • Criação e autenticação de conta;
  • Perfil de usuário com histórico de pedidos e endereços;
  • Carrinho de compras;
  • Escolha de frete e realização do pedido (no momento, sem pagamento);
  • Filtro de produtos por categoria e busca por termo;
  • Área administrativa.

Este projeto ainda está em desenvolvimento!.

O código-fonte está disponível no meu Github.

Instalação:

Para utilizar o projeto, basta que você tenha uma instalação do Java em sua máquina, pois o projeto foi compilado em um pacote .jar pelo Maven, dessa forma, basta apenas executar o arquivo e abrir o navegador no link: http://localhost:9000/swagger-ui/index.html.

Link para download do arquivo (Hospedado na Firebase Storage): Projeto Ecommerce v1.1

Observações:
- O projeto está rodando na porta 9000, caso esta estiver ocupada, não irá funcionar.

- No pacote do download, o banco de dados utilizado é o H2 ao invés do MySQL. Isso ocorre pela praticidade do banco H2, que permite que seja utilizado um arquivo como fonte de dados, dessa forma, facilitando a criação do pacote do projeto e também removendo a necessidade do usuário instalar e executar o MySQL na sua máquina.

- Caso tenha o Docker instalado em sua máquina, o projeto também disponibiliza um arquivo docker compose para fazer a execução dos containers do projeto com todas as suas dependências. O arquivo está disponível em: https://github.com/leonardorocio/ecommerce-project

Endpoints da API:

Endereço:

Método URI Descrição
GET /address Retorna todos os endereços no banco de dados
GET /address/{id} Retorna um endereço, identificado pelo parâmetro id
POST /address Cria um novo endereço
PUT /address/{id} Atualiza o endereço, identificado pelo parâmetro id
DELETE /address/{id} Deleta o endereço, identificado pelo parâmetro id

PATCH /users/{id} Atualiza o usuário, identificado pelo parâmetro id
