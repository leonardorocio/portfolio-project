Projeto Web: JML Musculação

Apresentação:

A disciplina de Projeto Integrador do Curso Técnico em Informática Integrado ao Ensino Médio do IFC - Campus Araquari propõe a criação de um sistema com tema de escolha de um grupo de alunos, sendo necessário que ao menos uma interação com banco de dados esteja presente por meio de um sistema de autenticação.

O projeto foi desenvolvido com a codificação do backend foi feita em Python 3.x com o framework web Django criando uma API por meio do Django REST Framework, enquanto o frontend da aplicação está sendo desenvolvido utilizando HTML, CSS, o framework de estilos responsivos Bootstrap 4 e Javascript, sendo utilizado o framework VueJS para a interface de usuário. Além disso, o banco de dados escolhido foi o SQLite gerenciado pelo próprio ORM do Django.

O código-fonte está disponível no meu Github.

Instalação:

O processo de instalação do projeto está descrito passo-a-passo no repositório do Github. Mas, resumindo, é necessário que se instale o Python 3, adicione ao PATH, abra a pasta "Backend" do projeto e utilize o comando python manage.py runserver. Por padrão ele irá executar na porta 8000, é importante que seja nesta porta. (O email do super usuário é admin@admin.com e a senha é admin). Feito isso, é possível acessar a parte administrativa (fornecida pelo Django) da API, para facilitar o uso no link: http://localhost:8000/admin.

Para o frontend, basta ter o NodeJS e o NPM instalados, então abra a pasta "frontend" do projeto e execute o comando npm run serve. É importante lembrar de deixar o backend executando também.v
